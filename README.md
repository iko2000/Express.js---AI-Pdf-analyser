# AI-PDF Analyzer

A simple Express server that uses AI to analyze and extract information from PDF documents.

## Features

- PDF file upload and processing
- AI-powered text extraction and analysis
- RESTful API endpoints for PDF operations
- Simple and intuitive interface

## Installation

```bash
# Clone the repository
git clone https://github.com/iko2000/Express.js---AI-Pdf-analyser/new/main?filename=README.md
cd ai-pdf-analyzer

# Install dependencies
npm install
